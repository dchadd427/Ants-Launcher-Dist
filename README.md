### Connection Issues?
If you can't connect to other players (especially in a strict network environment), you may need to configure port forwarding because we use **Direct P2P**.

-   **Protocol**: UDP
-   **Port Range**: `50000-50100`

See the full [NAT Traversal Guide](https://github.com/dchadd427/Ants-Launcher/blob/master/docs/NAT_TRAVERSAL.md) (in the source repo) for details.
